# Inventory Management System

A full-stack web application for managing inventory, sales, profit reports, and data analysis.  
Built with Django (backend) and React (frontend).

---

## 🚀 Features

- Upload Excel/CSV files for products and sales
- Track inventory and stock levels
- Sales analytics and profit reports
- Top selling items (Top N)
- Search functionality
- Optimized backend performance for large datasets

---

## 🛠️ Tech Stack

### Backend:
- Django
- Django REST Framework
- SQLite
- Pandas (for Excel processing)

### Frontend:
- React
- Axios
- Tailwind CSS / Bootstrap (if used)
